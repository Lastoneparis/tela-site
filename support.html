Tela Support

Tela lets you chat with any AI model using your own API keys. Here's how to get started and get help.

Getting started

  1. Open Settings → API Keys and paste a key from your provider (OpenAI, Anthropic, Google, DeepSeek, Groq, or OpenRouter).
  2. Tap the compose button to start a chat and pick a model.
  3. Your key stays in the device Keychain; messages go straight to the provider.

Frequently asked

Where do I get an API key? From your chosen provider's website (their API/developer dashboard). You pay the provider directly for usage.

Is my data private? Yes — see our Privacy Policy. We host nothing.

What does Premium add? PDF/document chat, web search, iCloud sync, a prompt library, and export.

How do I restore a purchase? Settings → Restore Purchases.
